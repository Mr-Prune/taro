### Image
图片组件
### Props

|     | 名称                   | 类型    | 默认值       | 说明                                                                                        |
| --- | ---------------------- | ------- | ------------ | ------------------------------------------------------------------------------------------- |
| √   | src                   | url  | ---      | iamge的地址                                                                              |
| √   | alt                   | String  | 'taro-img'      |                                                                               |
|     | model                 | cover、contain、stretch、 center | cover        | 图片的缩放类型                                                                    |

mode类型应如下

|  类型 | 描述 | 对应taro中的类型 |
| --- | ---------------------- | ------- | 
| cover |  保持宽高比缩小或放大，使得两边都大于或等于显示边界，居中显示 | widthFix |
| contain | 不保存宽高比，填充满显示边界 | scaleToFill | 
| stretch | 保持宽高比缩小或放大，使得两边都大于或等于显示边界，居中显示 | - |
| center | 居中，无缩放 | center |
